# SecurePass

Development notes

- Server: `server`
- Client: `client`

Quick start

1. Install client deps and run dev server:

```powershell
cd client
npm install
npm run dev
```

2. Optional: run the MCP test server added under `server/mcp-server`:

```powershell
cd server\mcp-server
npm install
npm run dev
```

Notes
- See `server/.env.example` and `server/mcp-server/.env.example` for env vars.
